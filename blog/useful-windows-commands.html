密碼最長使用期限

Windows 10 密碼最長使用期限 內定為 42 天, 假如此一設定對使用者造成困擾, 可以從系統管理者執行的 cmd 中, 輸入 secpol.msc 開啟 本機安全性設定, 開啟"帳戶原則"中的"密碼原則", 然後將"密碼最長使用期限"從內定的 42 天改為 0, 表示密碼永不過期.
